使用hexo搭建个人博客————建站

发表于 2019-10-12 更新于 2019-10-14 分类于 博客搭建 阅读次数： 评论数：

  现在又很多的平台，只要注册就能拥有自己的博客，例如CSDN、博客园、掘金等，然而这些博客样式千篇一律，但你又想与众不同，那就只能自己动手搭建一个属于自己的个人博客了。
  本文介绍了如何用hexo框架在github上搭建属于自己的博客，使用hexo你可以随意更换主题，更甚者可以自己制作主题（但对于不会代码的人来说有点小困难）并分享给别人使用。
以下仅是我自身的经验之谈，有错误的地方敬请指出

  1. 准备工作
    在建站之前请确保你已经安装以下应用程序：
  • Node.js (版本需不低于 8.6，建议使用 10.0 及以上版本)
  • git
    不懂安装的请自行百度，这里不再多作说明

本地调试

在项目的根目录运行以下命令

在浏览器中访问：http://localhost:4000 就可以看到你本定运行的页面了，并进行本地调试
